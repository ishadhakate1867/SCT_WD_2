# Calculator Web App

A responsive and interactive calculator web application built using **HTML, CSS, and JavaScript** as part of the **SkillCraft Technology Web Development Internship – Task 2**.

## 🚀 Features

- Perform basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Percentage (%) calculation
- Decimal number support
- Keyboard input support
- Backspace (⌫) functionality
- Clear (AC) button
- Error handling for invalid expressions
- Responsive and modern user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
